# supportBot

SupportBot — это простой чат-бот службы поддержки, созданный с помощью HTML, CSS и JavaScript.

## Цель проекта
Создать бота поддержки, который помогает пользователям получать ответы на частые вопросы.

## Возможности
- отвечает на вопросы по доставке
- объясняет способы оплаты
- сообщает условия возврата
- предоставляет контакты службы поддержки

## Использованные технологии
- HTML
- CSS
- JavaScript
- GitHub
- GitHub Pages

## Результат
В ходе работы был создан чат-бот поддержки, который отвечает на основные вопросы пользователей.
